# Intro ML project

A practice project exploring basic approaches to estimating rental prices based on property features. The goal is to train simple models and compare their results.

## Overview

This repository contains a Jupyter Notebook that demonstrates:

- Loading and exploring structured data
- Basic statistical analysis and visualization
- Handling categorical attributes
- Creating simple derived features
- Training and evaluating several baseline models
- Comparing model performance using standard error metrics

## Tools & Libraries

- Python 3
- Jupyter Notebook
- pandas, numpy – data handling
- matplotlib, seaborn – visualization
- scikit-learn – model implementation and preprocessing

## Contents

- `regression.ipynb` – main notebook with all analysis and experiments
- `data/` – folder containing the dataset 
- `README.md` – project description

##  What’s Covered

1. **Initial exploration** – understanding data structure, basic statistics, and feature types  
2. **Statistical analysis** – distribution checks, outlier detection, correlation overview  
3. **Feature transformations** – encoding categorical variables, generating polynomial features  
4. **Modeling** – implementing simple regression models, tree-based models, and reference baselines  
5. **Evaluation** – comparing models using error metrics to identify the most effective approach
